# Trigger Matrix — Research sessions

Read this file when session-review classifies the current session as **Research** (Read/Grep/WebSearch/Agent heavy, few writes).

| Signal | Recommended skill | Priority |
|--------|------------------|----------|
| Findings/discoveries not saved | mnemo:save | CRITICAL |
| No session summary created | mnemo:session | HIGH |
| New knowledge links to existing notes | mnemo:connect | MEDIUM |
| External articles / papers discovered | mnemo:save (as Source note) | HIGH |
| Multiple related findings surfaced | mnemo:save (as Molecule synthesizing them) | MEDIUM |
| PKM / vault patterns discovered (naming rules, note-type conventions, graph fixes) | mnemo:save (as Atom stating the rule) + update `references/` if it's a skill rule | HIGH |

Also check universal triggers in `triggers-universal.md`.
